# Документация проекта ProDvor

В этой директории собрана вся ключевая документация по проекту.

**Важно:** Единственным и неоспоримым источником правды об API бэкенда является спецификация `OPENAPI_SPEC.json`, которая автоматически синхронизируется с сервером. Остальные документы служат для обзора и пояснения.

## Основные документы

-   **`API_REFERENCE.md`**: Краткое руководство по основным принципам взаимодействия с API (авторизация, пагинация, ошибки) и примеры ключевых эндпоинтов.
-   **`OPENAPI_SPEC.json`**: Локальная копия официальной спецификации API. **Не редактировать вручную.**
-   **`OPENAPI_SYNC.md`**: Инструкция по синхронизации локальной спецификации с бэкендом и генерации TypeScript-типов.
-   **`DATABASE_SCHEMA_OVERVIEW.md`**: Высокоуровневый обзор основных сущностей базы данных и их связей.
-   **`AI_ASSISTANT_INTEGRATION_GUIDE.md`**: Правила и рекомендации по работе с AI-ассистентом в контексте этого репозитория.
