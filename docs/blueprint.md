# **App Name**: ProDvor Landing

## Core Features:

- Landing Page: Landing page showcasing ProDvor platform features.
- Benefit Sections: Sections for key benefits: AI-powered assistance, tournaments, and community building.
- Call-to-Action Buttons: Clear call-to-action buttons to encourage user registration ('Начать побеждать', 'Создать аккаунт').
- Step-by-Step Guide: A step-by-step guide for users to easily understand how to join the platform.
- Logo variation generator: Use a tool to generate variations of the logo of ProDvor for users who need assistance to create a team logo.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey energy and dynamism, reflecting the competitive spirit of the platform.
- Background color: A dark, desaturated blue (#1A202C) to provide a modern and immersive feel, contrasting with the bright primary color.
- Accent color: A contrasting orange (#FF8C00) to highlight call-to-action buttons and important elements.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern look suitable for both headings and body text.
- Use clean and modern icons to represent different features of the platform. Simple and intuitive, avoid excessive detail.
- Ensure a clear and intuitive layout. Content is easy to read and the structure is visually consistent.
- Subtle animations on button hover states.
